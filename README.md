## Jogo do Número Secreto
Um jogo interativo e responsivo onde o usuário tenta adivinhar um número secreto gerado aleatoriamente pelo sistema. O projeto conta com feedback visual e sonoro para cada tentativa.
Este projeto foi desenvolvido como parte do curso de Iniciante em Programação da Alura, com foco no aprendizado prático de lógica de programação e manipulação do DOM.

## Tecnologias Utilizadas

* HTML5: Estruturação da página e dos elementos de interface.
* CSS3: Estilização, layout responsivo e design visual.
* JavaScript (ES6): Lógica do jogo, manipulação do DOM, controle de tentativas e geração de números aleatórios.
* ResponsiveVoice JS: Integração de voz para leitura dos textos na tela (Acessibilidade).
